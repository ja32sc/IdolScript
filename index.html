IdolScript

A JavaScript-inspired language with an Oshi no Ko twist!

Features Examples Static Errors

Features

  • JavaScript-like syntax
  • Idol and Oshi no Ko themed keywords
  • String interpolation for easy combination
  • Future support for hidden variables (`secrets` keyword)
  • Themed error messages (coming soon!)

Examples

Functions

function addNumbers(a, b) {
  return a + b;
}
        
episode addNumbers(a, b) {
  encore a + b;
}

Static Errors

  • Undeclared Variable
  • Invalid `encore` Usage
  • Infinite Recursion Detection
