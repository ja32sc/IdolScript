IdolScript

A JavaScript-inspired language with an Oshi no Ko twist!

Features Examples Static Errors

Features

  • JavaScript-like syntax
  • Idol and Oshi no Ko themed keywords
  • String interpolation for string combination
  • Hidden variables (future secrets keyword)
  • Themed Error Messages (upcoming)

Static Errors

  • Undeclared Variable: Using a variable that hasn’t been declared.
  • Out-of-Scope Variable: Accessing a variable outside its scope.
  • Invalid `encore` (return) Usage outside a function.
  • Invalid `exitStage` (break) Usage outside a loop.
  • Invalid Operations Between Types.
  • Function Return Type Mismatch.
  • Wrong Number of Arguments.
  • Calling a Non-Function.
  • Infinite Recursion Detection.
  • Redeclaration of Constants (`idol`).
  • Invalid Array Operations.
  • Accessing Private (Backstage) Variables from Outside.
